# Bolet_api_gateway
Api gateway para el proyecto Bolet

Este repositorio esta hecho para realizar la conexión entre el frontend y el backend(microservicios) para el funcionamiento final de la pagína Bolet, este contiene los documentos necesario para realizar las acciones adecuadas del backend auth, java backend y category backend y adecuarlas a los resultados obtenidos en el frontend.

Las dependencias usados para este repositorio se encuentran en el documento package.json y package-lock.json

Las dependencias usadas son:
        "apollo-server":"^3.1.2",
        "apollo-datasource-rest" :"^3.1.1",
        "graphql":"^15.5.1",
        "lodash":"^4.17.21",
        "node-fetch":"^2.6.0"


para installar las dependencias/paquetes se usa el comando npm install package*.json
